Frontend Web Development

This is Week 4's Exercise

Descriptions of Aim and Content

The Front-end Web Development module aims to equip you, the student, with the essential skills and knowledge to create
visually appealing and interactive web pages. You will learn about the key technologies used in front-end web development,
including regular expressions, HTML, CSS, and JavaScript. The module will cover basic and advanced topics in each of these
areas, with a focus on creating responsive web pages that adapt to different screen sizes and devices. By the end of the
module, you will have a solid understanding of front-end web development and be prepared for further study or
professional practice in the field.

Indicative Content

  • Regular Expressions:
    • Introduction to regular expressions and their use in searching and manipulating text data within
      web applications
    • Creating regular expressions to match patterns in text data
    • Advanced regular expressions techniques, such as look-around and back-references
    • Using regular expressions to validate user input and sanitize data
  • Basic and Advanced HTML/CSS:
    • Introduction to HTML and its role in creating web pages
    • Basic HTML tags and their use in structuring web pages
    • Creating and formatting web page content using CSS
    • Responsive design techniques using media queries and flexbox
    • Advanced CSS techniques, such as animations and transitions
  • Basic and Advanced JavaScript:
    • Introduction to JavaScript and its use in creating interactive web applications
    • JavaScript syntax, data types, and control structures
    • Manipulating the Document Object Model (DOM) using JavaScript
    • Event handling and creating event listeners
    • Advanced JavaScript techniques, such as closures and prototypes
  • Web Scraping using jQuery:
    • Introduction to web scraping and its uses in data extraction
    • Using jQuery selectors to extract data from web pages
    • Processing and manipulating extracted data using JavaScript
    • Storing extracted data in a structured format, such as JSON or CSV

Learning Outcomes

Knowledge and Understanding

Upon completion of this module, you should be able to:

  • Understand and apply regular expressions to search and manipulate text data within web applications.
  • Design and develop basic and advanced HTML/CSS web pages that are responsive and accessible.
  • Develop interactive web applications using basic and advanced JavaScript programming concepts, including
    DOM manipulation and event handling.
  • Extract data from websites using web scraping techniques and jQuery.

The course material's goal is to give you a thorough grasp and comprehension of the foundational ideas and technologies
utilized in front-end web development. This includes the ability to use regular expressions to manipulate text and data,
create valid HTML and CSS code, and write basic and advanced JavaScript programs. You will also be able to use
responsive design principles to make sure that your websites are usable and aesthetically pleasing across a variety
of devices. Ultimately, the aim is to get you to become well-versed in front-end web development and equipped to
produce web pages that are up to par with industry standards.
